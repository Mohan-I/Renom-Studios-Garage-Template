# Renom Studios Garage Template

A high-performance, modern web template designed for automotive studios and garages. This project focuses on a premium UI/UX experience with custom cursor interactions and an atmospheric "Retro-Tech" visual style.

-----

## 🚀 Features

  * **Custom Interactive Cursor:** A synchronized, spinning custom cursor that replaces the default browser pointer for an immersive feel.
  * **TV-Frame Aesthetic:** A fixed "Old TV" overlay with 89% opacity grain and vignette borders to create a vintage CRT monitor illusion.
  * **Smooth Navigation:** Integrated `scroll-behavior: smooth` for seamless transitions between sections.
  * **Optimized Performance:** Built with clean HTML and CSS, utilizing modern properties like `will-change` for high-frame-rate animations.
  * **Desert Monochromatic Palette:** A sophisticated HSL-based color system inspired by minimalist desert landscapes.

-----

## 🛠️ Technical Stack

  * **HTML5:** Semantic structure for SEO and accessibility.
  * **CSS3:** Advanced positioning (`fixed`, `absolute`), HSL variable mapping, and `@keyframes` animations.
  * **JavaScript:** Vanilla JS for viewport-relative mouse tracking (`clientX`/`clientY`).

-----

## 🎨 Color Palette

[cite_start]The project uses a custom HSL palette to maintain a consistent professional aesthetic[cite: 3, 5, 11]:

  * **Background:** `hsl(0, 0%, 96%)`
  * **Primary Text:** `hsl(0, 0%, 15%)`
  * **Accent (Highlights):** `hsl(0, 0%, 90%)`
  * **Shadows:** `hsla(0, 0%, 0%, .15)`

## 📂 Getting Started

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/Mohan-l/Renom-Studios-Garage-Template.git
    ```

2.  **Open the project:**
    Simply open `index.html` in your preferred browser.

3.  **Customization:**
    Modify the `:root` variables in the CSS file to adjust the theme colors or the TV static intensity.
